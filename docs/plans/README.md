# 功能方案 / 计划

> 共 16 篇,按「层」组织。底层能力是地基(服务所有上层);媒体库线管本地;探索线管全网;视频获取横跨两线。

## 建议实施顺序

1. **地基先行**:反爬工具箱 → 跨语言别名/规范化 + 番号识别增强 + 有码无码分轨 → 字段级跨源融合。(这些决定刮削质量与稳定性,上层全依赖)
2. **维度独立化**:studio/series/director 建表 + 番号前缀→系列(多维度框架阶段 1)。
3. **演员模块**:档案(JavBus star + gfriends/xslist 头像名)+ 作品全集 + 本地匹配。
4. **探索骨架 + 多维度**:独立探索入口 + 分面浏览 + 本地标记/过滤。
5. **视频获取**:快路径(含 MissAV surrit CDN)+ WebView 池;在线预览;磁力下载闭环。
6. **媒体库增强**:封面获取 + NFO 标准化/独立存储 + 库健康诊断 + 标准/非标准库。
7. **体验补充**:观看进度、演员新作订阅、MetaTube sidecar。
