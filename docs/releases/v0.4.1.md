# JAVM v0.4.1

> `v0.4.0` → `v0.4.1`

## 新增

- **全局日志系统**：集成 Tauri 官方日志插件（`tauri-plugin-log`），前端和后端的所有错误、警告统一持久化到日志文件，支持导出和诊断。
  - 前端全局捕获 Vue 错误、`window.error`、`unhandledrejection`，自动写入日志。
  - 后端全部业务模块（扫描、刮削、下载、媒体处理、数据库、分析等）升级为结构化日志（`event=...` 格式），便于聚合排查。
  - Panic hook 捕获后端崩溃信息并写入日志。
- **日志导出与诊断**：设置页新增"打开日志目录"和"导出日志"功能，导出包含日志文件和 `diagnostic.json`（含系统信息、近期异常摘要、事件级问题统计）。
- **扫描摘要**：目录扫描不再因单个文件失败中断，扫描结束后展示成功/失败文件数，批量同步时显示失败目录数。

## 优化

- 扫描目录时单文件处理失败会跳过并继续后续文件，不再导致整个扫描中断。
- 对话框组件统一补全 `aria-describedby` 属性，消除 Reka UI 的无障碍警告。

## 修复

- 移除重复的全局"扫描完成"通知，改为每次操作独立显示含成功/失败计数的摘要提示。
