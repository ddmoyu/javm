# JAVM v0.2.2

## 概览

- 发布范围：`v0.2.1..HEAD`
- 新增应用更新能力，支持检查并安装新版本。
- 新增发布说明获取能力，可从 GitHub Release API 拉取指定版本说明。
- 发布工作流同步增强，支持在发布流程中输出发布说明内容。

## 新增

- 新增 updater 模块，集中管理应用更新相关逻辑与调用入口。
- 实现应用更新检查与安装流程，降低手动升级成本。
- 增加按版本获取发布说明的能力，便于在应用内查看更新内容。

## 优化

- 优化发布流程配置，补齐发布说明在自动化流程中的输出能力。
- 更新后端模块组织，提升更新能力相关代码的可维护性。

## 修复

- 本次版本未包含独立的问题修复项，重点为更新能力建设与流程完善。
