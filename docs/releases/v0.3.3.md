# JAVM v0.3.3

> `v0.3.1` → `v0.3.3`

## 新增

- 新增视频管理功能模块，支持视频文件的元数据处理、移动、复制及删除操作
- 扫描时自动为无封面的视频截帧生成封面
- 支持本地 TS 格式视频文件播放（通过 HLS.js 合成 M3U8）
- 设置页新增邀请码一键复制功能和问题反馈快捷入口
- 数据源管理新增平均丰富度得分和累计刮削次数展示

## 优化

- 引入自适应并发控制器，根据 CPU 核心数和系统负载动态调整扫描与截帧的并发上限，提升整体性能
- 增强 Cloudflare 验证检测逻辑，避免内容丰富页面被误判为验证页；合并探测与拦截脚本，降低验证期间干扰
- 增加 HTTP 4xx 错误检测，优化 WebView 回退机制
- 设置界面引入折叠组件，改善信息层级与浏览体验

## 修复

- 新增路径校验功能，防止路径穿越导致的安全风险
- 修复 tsconfig.json 中已弃用配置导致 CI 构建失败的问题，确保 Windows x64 安装包正常生成
