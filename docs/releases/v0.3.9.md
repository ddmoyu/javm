# JAVM v0.3.9

> `v0.3.8` → `v0.3.9`

## 版本概览

- 本次更新主要补强媒体库交互，新增“点击封面直接播放”设置，并补上按目录筛选视频的能力。
- 同时整理了部分依赖与项目指令配置，提升后续维护与稳定性。

## 主要更新

### 媒体库播放交互

- 设置页新增“点击封面直接播放”选项，可在“点击封面直接播放”和“点击封面打开详情”之间切换。
- 卡片模式和列表模式的封面点击行为已统一，封面区域会按你的设置执行播放或打开详情，其余区域仍保持查看详情。

### 目录筛选

- 媒体库筛选面板新增目录下拉，可以只查看指定目录下的视频。
- 目录筛选同时支持子目录内容，适合在多库场景下快速聚焦某个目录分组。

### 稳定性与维护

- 更新了部分 Rust 依赖版本，改善兼容性与稳定性。
- 补充并整理项目指令与配置文件，统一版本发布、协作和实现约束。

## 使用说明

- 如果希望单击封面就播放视频，可前往设置页开启“点击封面直接播放”；关闭后，点击封面会恢复为打开详情。
- 需要只看某个目录内容时，可在媒体库的筛选面板里选择目标目录；切回“全部目录”即可恢复全量显示。
