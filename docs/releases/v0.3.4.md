# JAVM v0.3.4

> `v0.3.3` → `v0.3.4`

## 版本概览

- 下载链路继续补强，新增已完成任务同步入库能力，下载完成后的结果可以更快回到媒体库管理流程。
- 下载管理器支持运行时调整并发数，修改“同时下载数”后无需重启即可对后续任务生效。
- 视频链接查找与批量添加任务的容错进一步放宽，减少因为链接后缀或预检查过严导致的误判。

## 主要更新

### 下载体验

- 新增“已完成下载任务同步到媒体库”能力，便于把下载结果补录到库中继续管理。
- 下载管理器增强了任务停止与应用退出时的清理逻辑，减少残留状态不一致的问题。
- 设置中的下载并发数现在可以动态同步到运行中的下载管理器，后续排队任务会按新的并发上限继续执行。

### 链接识别与查找

- 视频链接查找器增强了 WebView 模式下的稳定性，统一了请求侧 User-Agent，并优化了 Cloudflare 场景下的处理流程。
- 批量添加下载任务时，前端不再仅凭固定后缀严格拦截候选链接，更多链接会交由下载器实际判断是否可用。

## 使用说明

- 如果你会在设置里调整“同时下载数”，本版本开始保存后会实时影响后续启动的下载任务。
- 已完成的下载任务如需补录到媒体库，可使用对应的同步能力进入统一管理流程。

## 风险与说明

- 下载并发数在运行时调小时，不会强行中断已经在执行的任务；会在当前任务逐步结束后收敛到新的并发上限。
