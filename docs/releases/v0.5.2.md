# JAVM v0.5.2

> `v0.5.1` → `v0.5.2`

## 新增

- 默认刮削网站新增「自动（最高分）」：自动选用累计得分最高的数据源，无需手动指定。
- 新增下载源管理：可单独启用/禁用各视频源，并按抓取成功情况自动排序常用源靠前。
- 资源链接查找新增正片识别：把最可能是正片（完整影片）的链接排到列表最前。

## 优化

- 资源链接查找改用 1080p 视口加载，更多站点会返回更高分辨率的视频。
- 「遇到验证是否弹窗」统一由设置中的「HTTP 失败回退 WebView」开关控制，关闭后即使遇到验证也不再弹出窗口。
- 刮削结果排序更准：按封面预览数量、以及你手动选用过的数据源优先展示。
- 「资源刮削」更名为「刮削」。

## 修复

- 媒体库：从其它页面切回时不再丢失滚动位置，也不再出现需手动滚动一下才显示内容的空白。
- 下载管理：删除任务会真正停止下载进程并清理残留文件，不再留下杀不掉的后台进程或删不掉的目录；重命名、修改保存路径时保留已下载进度可续传；「开始」只对排队/失败/已暂停的任务生效，不再重置正在进行的任务。
- 下载并发：设置同时下载多个时现在会真正并行，不再只下载 1 个。
- 下载任务重命名输入框中，文件名里的下划线不再被显示成空格。
- 媒体库广告过滤相关问题修复。
