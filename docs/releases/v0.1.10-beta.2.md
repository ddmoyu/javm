# JAVM v0.1.10-beta.2

## 概览

- 发布范围：`v0.1.10-beta.1..HEAD`
- 修复应用内更新发布链路中的公钥注入问题，避免 macOS 与 Linux 在生成 updater 产物时因读取占位公钥而构建失败。
- 清理桌面端发版相关的 Rust 编译 warning，减少构建日志噪音，便于后续排查真正的发布错误。

## 新增

- 无面向最终用户的新功能。

## 优化

- **发布工作流配置修正**：构建阶段会显式注入 updater 所需的真实公钥与更新地址，不再依赖开发环境占位配置参与正式打包。
- **构建日志更干净**：清理深链注册与下载管理中的无用 import、无用变量，降低 CI 日志中的干扰信息。

## 修复

- 修复 macOS 与 Linux 在生成 updater 产物时，因错误读取开发占位公钥导致 `failed to decode pubkey` 并中断发布的问题。
- 修复下载管理器中平台可执行文件候选路径函数的未使用变量 warning。
- 修复桌面端深链初始化路径中的未使用 import warning。

## 兼容性与迁移

- 无特殊操作要求。
- 本版本仍依赖 GitHub Actions 中已配置的 updater 公钥、签名私钥与密码；若这些仓库变量或 Secrets 缺失，应用内更新构建仍无法成功。

## 已知问题

- 暂无新增已知问题。
