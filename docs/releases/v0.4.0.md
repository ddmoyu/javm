# JAVM v0.4.0

> `v0.3.9` → `v0.4.0`

## 版本概览

- 本次更新聚焦视频封面链路，补强了仅保存海报路径时的封面解析与展示逻辑。
- 下载完成后的后处理流程新增截图封面能力，进一步降低封面缺失时的手工处理成本。
- 同时更新了部分前端与 Rust 依赖版本，提升整体兼容性与稳定性。

## 主要更新

### 封面处理改进

- 数据库更新逻辑现在支持仅保存海报路径，减少封面元数据不完整时的显示问题。
- 封面路径解析会优先使用海报路径，媒体库卡片、列表项和详情弹窗的封面显示逻辑已同步调整。
- 相关扫描、媒体命令和视频命令链路已一并更新，封面数据在不同入口下的表现更一致。

### 下载后处理增强

- 下载完成后的后处理流程新增截图封面能力，在常规封面资源不足时可补充可用封面素材。

### 稳定性与维护

- 更新了 Reka UI 及部分 WebAssembly、系统依赖版本，改善兼容性与运行稳定性。

## 使用说明

- 如果部分旧数据只保留了海报路径，升级后媒体库会优先使用该路径解析封面，无需额外手工修复。
- 新下载任务完成后，如原始封面资源不足，后处理流程会按新的封面策略补充展示素材。

## 风险与说明

- 本次发布包含封面字段读写与展示逻辑调整；如你本地存在依赖旧封面字段的外部流程，建议升级后先抽样检查几条视频数据的展示结果。
