# JAVM v0.1.10-beta.6

## 概览

- 发布范围：`v0.1.10-beta.5..HEAD`
- 修复数据刮削弹窗中的预览图类型处理问题，避免生产构建在 TypeScript 检查阶段失败。
- 本版本主要用于验证 `tauri build` 与 GitHub Actions Release 流程是否已恢复正常。

## 新增

- 无新增功能。

## 优化

- **预览图数据收敛更明确**：预览图列表改为直接生成稳定的 `PreviewImage[]`，减少空值参与后续模板渲染和图片预览流程的可能性。

## 修复

- 修复数据刮削弹窗中预览图列表的类型推断问题，避免 `bun run build` 在 `ScrapeDialog.vue` 的类型检查阶段报错。
- 修复由预览图空值联合类型引发的构建阻塞问题，避免测试版发布时所有平台都在 `tauri build` 阶段统一失败。

## 兼容性与迁移

- 无特殊操作要求。

## 已知问题

- 暂无新增已知问题。
